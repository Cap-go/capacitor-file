# Changelog

## [7.0.0] - Initial Release

### Added
- Full file system operations API compatible with cordova-plugin-file
- Support for reading and writing files (text and binary)
- Directory operations (create, read, delete)
- File metadata and stat operations
- Copy, move, and rename operations
- Existence checking
- URI resolution
- File truncation
- Get free disk space
- Get known directories
- iOS and Android native implementations
- Web implementation using IndexedDB
